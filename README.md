# CurrentWeather

Консольное приложение для запроса текущей погоды через [OpenWeatherMap API](https://openweathermap.org/api): геокодинг города, получение данных по координатам, кэш последнего успешного ответа и повторные попытки при временных сбоях сети или ограничении по частоте запросов.

## Возможности

- Запрос по названию города или по широте и долготе.
- Понятные сообщения об ошибках (неверный ключ, город не найден, сетевые сбои) без вывода traceback в обычном режиме.
- Сохранение последнего успешного ответа в `weather_cache.json` (город, координаты, время получения, JSON погоды). При сетевой ошибке можно вывести кэш, если данным меньше 3 часов.
- До 4 попыток HTTP-запроса с паузами 1 с, 2 с, 4 с при кодах 429 и 5xx и при сетевых исключениях.

## Требования

- Python 3.9+ (используются аннотации вида `dict[str, ...]`).
- Бесплатный API-ключ OpenWeatherMap.

## Установка

1. Клонируйте или скопируйте каталог проекта.

2. Создайте виртуальное окружение (рекомендуется):

   ```bash
   python -m venv .venv
   ```

3. Активируйте окружение:

   - Windows (PowerShell): `.venv\Scripts\Activate.ps1`
   - Linux/macOS: `source .venv/bin/activate`

4. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

5. Создайте в корне проекта файл `.env` и укажите ключ:

   ```env
   API_KEY=ваш_ключ_openweathermap
   ```

   Ключ можно получить после регистрации на https://openweathermap.org/api.

## Запуск

Точка входа — файл `cli.py`. Запускайте из каталога проекта:

```bash
python cli.py
```

## Использование

После запуска выберите режим:

| Ввод | Действие |
|------|----------|
| `1` | Запрос по городу: введите название — выводится строка «Погода в г.…: …°C, …». |
| `2` | Запрос по координатам: введите широту и долготу (допускается запятая как разделитель дробной части). |
| `0` | Выход из программы. |

При ошибке сети после исчерпания повторов, если в кэше есть свежие данные (младше 3 часов), программа предложит вывести их.

## Структура проекта

| Файл | Назначение |
|------|------------|
| `cli.py` | Точка входа: меню, форматирование вывода, сценарий с кэшем при сбоях. |
| `api_client.py` | HTTP-запросы к OpenWeatherMap, ретраи. |
| `storage.py` | Чтение и запись `weather_cache.json`, проверка «свежести» кэша. |

Файл `weather_cache.json` создаётся автоматически после успешного запроса.
